# SEO Assets

Coloque nesta pasta os arquivos abaixo para completar o SEO visual:

1. `og-default.jpg` (1200x630) - imagem para compartilhamento Open Graph/Twitter.
2. `apple-touch-icon.png` (180x180) - icone para iOS.
3. `icon-192.png` (192x192) - icone PWA.
4. `icon-512.png` (512x512) - icone PWA.
5. `icon-maskable-512.png` (512x512) - icone maskable PWA.
6. `logo-512.png` (512x512 recomendado) - logo para Schema.org (Organization).

Observacoes:

- Use nomes exatamente iguais para evitar quebrar referencias.
- Prefira fundo limpo e boa legibilidade para o `og-default.jpg`.
